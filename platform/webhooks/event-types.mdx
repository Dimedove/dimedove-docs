---
title: Event Types
description: Learn about available webhook event types and their data
---

Available webhook event types:

- **chat.started** - When a conversation begins
- **chat.ended** - When a conversation completes
- **chat.human_required** - When human assistance is needed
- **contact:insight.created** - When new contact insight is generated
- **contact:insight.updated** - When contact insight is modified
- **contact:attribute.created** - When new contact attribute is added
- **contact:attribute.updated** - When contact attribute is modified
- **task.completed** - When an agent task is finished
