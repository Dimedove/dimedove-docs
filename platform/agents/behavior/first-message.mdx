Set up the opening message your agent will use when starting conversations with visitors.

## Message Configuration

Create a welcoming and informative first message that:

- **Introduces the agent**: Let customers know they're talking to an AI assistant
- **Sets expectations**: Explain what kind of help is available
- **Offers guidance**: Suggest common questions or topics
- **Maintains brand voice**: Align with your company's communication style
