Find answers to the most common questions about Dimedove.

<AccordionGroup>
  <Accordion title="What is Dimedove?">
    Dimedove is an AI-powered customer engagement platform that helps businesses qualify leads and deliver personalized experiences through intelligent agents. Configure your agents to match your brand, deploy them across multiple channels, and manage all conversations from a unified inbox.
  </Accordion>

  <Accordion title="What channels does Dimedove support?">
    Dimedove supports a growing range of channels including a website messenger widget, standalone forms, email, telephony (voice and SMS), Slack, and socials. All conversations across channels are managed from a single unified inbox.
  </Accordion>

  <Accordion title="Can I customize my agent's behavior and personality?">
    Yes. You can configure your agent's identity, instructions, personality, tone, first message, verbosity, human fallback behavior, and more. See our [Agent Configuration](../platform/agents/overview) guide for details.
  </Accordion>

  <Accordion title="What integrations are available?">
    Slack is fully available today. Many more integrations, including HubSpot, Salesforce, Shopify, Stripe, Zapier, and others, are coming soon. Check the [Integrations](../integrations/slack) tab for the latest.
  </Accordion>

  <Accordion title="Does Dimedove have an API?">
    Yes. The Dimedove API lets you programmatically create conversations, send messages, and receive AI agent responses. We also support [webhooks](../developer/webhooks/overview) for real-time event notifications and [MCP server hosting](../developer/mcp/overview) for connecting agents to external AI tools. See the [API Reference](/api-reference/overview) for details.
  </Accordion>

  <Accordion title="How do I install the agent on my website?">
    Add the JavaScript widget code to your website using your agent ID and publishable key. See our [Installation Guide](../platform/agents/install) for step-by-step instructions.
  </Accordion>

  <Accordion title="How do workflows work?">
    Workflows let you connect platform events to automated actions. Configure triggers (like a new conversation starting or a contact being updated) and define the actions that follow. See the [Workflows](../platform/workflows/overview) documentation to get started.
  </Accordion>

  <Accordion title="How do I contact support?">
    For support questions, reach out to support@dimedove.com. If you need onboarding help or want to speak with the founder directly, [book a call with Felix](https://cal.com/felixsimard/onboarding).
  </Accordion>
</AccordionGroup>
